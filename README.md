# 慧眼识植 (FloraInsight) - AI 智能植株管家

## 项目背景

针对现代都市人群"想养花但养不活"的痛点，开发一款基于移动端（App/H5/小程序）+ 计算机视觉 (CV) + 大语言模型 (LLM) 的智能应用。

## 核心功能

- 📷 拍照识别植物品种
- 🦠 病害诊断与识别
- 💧 个性化养护建议
- ⏰ 智能浇水提醒

## AI 技术结合

- **计算机视觉**：识别植物品种和病虫害
- **大语言模型**：提供拟人化的养护知识问答

## 技术栈

### 待选技术
- **前端**：Flutter / React Native / Uni-app（移动端）
- **后端**：Node.js / Python / Go
- **AI 模型**：
  - 图像识别：YOLO / EfficientDet / ResNet
  - LLM：GPT-4 / DeepSeek / Ollama（本地部署）
- **数据库**：MySQL / MongoDB / PostgreSQL

## 项目结构

```
FloraInsight/
├── frontend/          # 移动端应用
├── backend/           # 后端服务
├── ai/                # AI 模型服务
│   ├── cv/            # 计算机视觉模型
│   └── llm/           # 大语言模型
├── docs/              # 文档
└── scripts/           # 工具脚本
```
